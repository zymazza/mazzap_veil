# Pure-Python deps for the pipeline + MCP/chat server, pip-installable on any
# platform:
#   mcp     — the MCP server (scripts/mcp_server.py) and chat bridge
#   pyproj  — in-Python WGS84 <-> projected-CRS transforms
#   Pillow  — raster layer rendering (atlas PNGs, survey package basemap)
#
# GDAL (the `osgeo` bindings) and numpy are NOT installed by pip here: GDAL's
# Python bindings must match the system GDAL, and the scripts use GDAL's array
# support (ReadAsArray/WriteArray), which is tied to a matching numpy. Get them
# from the OS package manager, or just use the container (Dockerfile /
# docker-compose.yml), whose OSGeo base image ships GDAL + bindings + numpy
# pinned together by GDAL_VERSION.
#
# The first-run AOI builder also uses the PDAL command-line tool when available
# to turn USGS 3DEP LiDAR Point Cloud LAZ tiles into DSM/DTM rasters. Without
# PDAL, LiDAR is skipped and the twin still builds from DEM/NAIP/LANDFIRE.
mcp>=1.20,<2
pyproj>=3.6,<4
Pillow>=10,<12
